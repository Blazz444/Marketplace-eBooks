# Ebook225 Marketplace

Une plateforme de marché pour vendre, acheter et télécharger des livres numériques en Côte d'Ivoire.

## ✨ Fonctionnalités

- **Catalogue d'eBooks** : Parcourez et achetez des livres numériques
- **Inscription vendeur** : Devenez vendeur et publiez vos propres eBooks
- **Gestion utilisateur** : Créez un compte et connectez-vous
- **Dashboard vendeur** : Suivi de vos ventes et gains
- **Intégration Firebase** : Synchronisation cloud et authentification
- **Support multilingue** : Interface en français

## 🏷️ Prix

- Réussir en Business : 5 000 FCFA
- Guide Python Débutant : 7 500 FCFA
- Concours Fonction Publique : 3 000 FCFA

## 🚀 Démarrage rapide

1. Ouvrez `marketplace_ebooks_complete_site.html` dans votre navigateur
2. Parcourez les eBooks disponibles
3. Inscrivez-vous comme vendeur ou créez un compte utilisateur

## 🔧 Configuration Firebase

Pour activer les fonctionnalités cloud :

1. Créez un projet sur [Firebase Console](https://console.firebase.google.com)
2. Récupérez :
   - Votre **API Key**
   - Votre **Project ID**
   - Votre **Database URL**
3. Entrez ces informations dans la section "Configuration Firebase"
4. Cliquez sur "Connecter Firebase"

## 💳 Paiement

Les paiements sont traités en FCFA. La plateforme applique une commission de 20% sur les ventes.
